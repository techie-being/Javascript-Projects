# 🌦️ Weather Page

A simple and clean weather web app that shows real-time weather data for any city using the `wttr.in` API.

## 🚀 Features
- Search any city 🌍
- Get current temperature 🌡️
- See humidity and weather conditions 🌧️
- Responsive and user-friendly UI 📱
